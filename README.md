# Terraform

## Run Terraform enviroment

```bash
make playground
```

Stop enviroment

```bash
make stop
```

## Docker example

Build an image with nginx and run a container.

In */projects/docker*

See the nginx in

```
http://localhost:3000
```

### Run assistant

```bash
$ ./run.sh
```

## Sequence

Go to the project folder

- terraform init
- terraform plan
- terraform apply

Terminate 

- terraform destroy
